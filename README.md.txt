Odontirx is a full-stack Electronic Dental Record (EDR) system with AI-based oral cancer risk prediction.
This project demonstrates how clinical workflows can be digitized and enhanced using modern web
technologies and machine learning.

## Tech Stack
- Frontend: React
- Backend: Node.js / Express
- Database: MySQL
- AI/ML: Python (Oral Cancer Risk Prediction)

## Project Structure
Odontirx-MiniEDR/
  mini-edr-frontend/
  mini-edr-backend/
  ML-Oral-Cancer-Prediction/
  database/
    schema.sql

## Setup Overview
1. Create a MySQL database.
2. Run `database/schema.sql` to create tables.
3. Start backend:
   - cd mini-edr-backend
   - npm install
   - npm start
4. Start frontend:
   - cd mini-edr-frontend
   - npm install
   - npm start

This project is built for learning and using synthetic data only.
